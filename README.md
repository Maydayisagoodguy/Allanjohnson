# MN Moopen Geofencing Demo

Prototype HTML pages for petrol pump geofencing, including:

- CA Ably live tracker
- Supervisor Ably live dashboard
- Boundary and coordinate sketches
- Simple phone geofence test
- Satellite/KML reference files

For the live demo, deploy this folder as a static site and open:

- `supervisor-ably-dashboard.html` on the supervisor laptop
- `ca-ably-tracker.html` on the CA phone
